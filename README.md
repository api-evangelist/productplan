# ProductPlan

ProductPlan is a road mapping software platform with a REST API for creating and managing roadmaps, features, goals, OKRs, integrations with Jira, GitHub, Slack, and Trello, and sharing plans with stakeholders. The platform covers the full product lifecycle: strategy, discovery, roadmapping, and launch management.

## APIs.json

The `apis.yml` file in this repository is an [APIs.json](https://apisjson.org) index conforming to specification version 0.19. It catalogs the ProductPlan REST API v2 and links to supplementary artifacts covering plans/pricing, rate limits, and FinOps guidance.

## Repository Structure

```
apis.yml                          # APIs.json 0.19 index
plans/
  productplan-plans-pricing.yml   # API Commons Plans 0.1
rate-limits/
  productplan-rate-limits.yml     # API Commons Rate Limits 0.1
finops/
  productplan-finops.yml          # FinOps Framework 1.0 FOCUS-aligned
```
